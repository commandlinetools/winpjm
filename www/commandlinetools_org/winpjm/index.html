WinPJM

Windows command line project management tool.

Purpose

Makes it easy to switch between projects when using the Windows command shell. Each project exists in its own directory and changing the current project sets up environment variables for that project. For example, two software development projects can use different versions of Java or ant.

This tool is not limited to supporting software development projects; it can simply help you remember a list of projects and which directories each project's files are in. It is also easy to list all projects, quickly change projects, add new projects, edit the setup for an existing project, remove old projects, and show the current project.

Commands

Frequently Used

  • chpj name - changes the current project; runs its setup script
  • lspj - lists projects

Occasionally Used

  • adpj name path - adds a project
  • edpj name - edits the setup script for a project
  • hlpj - help, shows commands
  • shpj - shows the current project name and path
  • rmpj name path - removes a project

Rarely Used

  • inpj - installs the project switching script
  • unpj - uninstalls the project switching script
